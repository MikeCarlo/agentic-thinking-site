## Key Takeaways

- ▸ The hosts frame Rayfin as the biggest Power BI/Fabric app-building change since Power BI Desktop, especially for agent-assisted development.
- ▸ The live flow is concrete: enable App Items in Fabric admin settings, run `npm create rayfin@latest`, develop with `npm run dev`, then deploy with `npx rayfin up`.
- ▸ Rayfin projects compile to simple web artifacts (HTML/CSS/JS), and apps are exposed via direct URLs with Microsoft sign-in.
- ▸ Skill placement matters: if `.agents/skills` lives in a subfolder, VS Code chat may miss it unless you open the scaffolded app folder as workspace root.
