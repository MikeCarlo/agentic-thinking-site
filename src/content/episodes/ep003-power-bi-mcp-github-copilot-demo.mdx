## Summary

A live demo episode showing the integration of Power BI with Model Context Protocol (MCP) and GitHub Copilot. Mathias and Mike demonstrate how MCP servers can bridge the gap between AI coding assistants and BI tooling, enabling context-aware AI development workflows.

## Key Takeaways

- ▸ MCP provides a standard protocol for AI tools to access external data sources
- ▸ An MCP server for Power BI exposes model metadata (tables, columns, measures, relationships)
- ▸ GitHub Copilot + MCP = context-aware DAX generation (no hallucinated table names)
- ▸ The MCP server reads from TMDL source files, staying in sync with version control
- ▸ This pattern works with any MCP-compatible AI tool, not just Copilot
