## Summary

Episode 2 features a live demo of [BIBuddy.ai](https://bibuddy.ai/) — an AI-powered companion for Power BI development built by Matthias. Inspired by feedback from a Power BI conference where many analysts had never opened an IDE, BIBuddy lowers the barrier by bringing agentic AI assistance directly into a familiar UI. Mike and Matthias walk through connecting BIBuddy to a local PBIP file and a Fabric workspace, generating DAX with full model context, and using the skills system to encode organizational best practices.

## Key Takeaways

- ▸ BIBuddy bridges the gap between AI tooling and analysts who have never used an IDE
- ▸ It connects to local PBIP projects or live Fabric workspaces via MCP
- ▸ The skills system lets teams encode DAX patterns and naming conventions as reusable prompts
- ▸ Bring-your-own-key: you supply your own API key so your data never leaves your control
- ▸ DAX generation is context-aware — the agent inspects your actual model before writing measures
